# Nova Folha - Diário de Crescimento

A personal growth journal application designed to nurture hope, reflection, and inner strength. Built with Next.js 16 and Cosmic CMS.

## Features
- **Journal Timeline:** View your journey of growth with visual indicators for feelings and inner weather.
- **Rich Reflections:** Write and read entries with full Markdown support.
- **Inspiration Board:** Collect and display quotes that resonate with your journey.
- **Visual Metaphors:** "Inner Climate" tracking using the specific metaphors from your content model (Sun, Gentle Rain, etc.).
- **Responsive Design:** Beautiful on mobile and desktop.

## Technologies
- **Framework:** Next.js 16 (App Router)
- **CMS:** Cosmic CMS
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Typography:** React Markdown & Tailwind Typography
- **Language:** TypeScript

## Getting Started

1.  **Install dependencies:**
    ```bash
    bun install
    ```

2.  **Set up environment variables:**
    Create a `.env.local` file with your Cosmic credentials.

3.  **Run the development server:**
    ```bash
    bun run dev
    ```

4.  **Open your browser:**
    Navigate to [http://localhost:3000](http://localhost:3000)

## Deployment

This application is ready for deployment on Vercel. Connect your repository and add the environment variables in the Vercel dashboard.
